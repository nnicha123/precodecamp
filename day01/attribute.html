tag attribute explanation
h
p title When put mouse over paragraph, title will appear
style Used for styling e.g. font size, align...
div class Set class names for element (use later)
id Set id name for element (use for styling)
img src For linking to URL of image file
alt Set alternate text when image not showing
width Set width of image
width Set height of image
a href Set reference URL, where link goes to
target Set where to open linked document e.g. _blank -> open in new window/tab
download Target will be downloaded with link is clicked
ul type Defines marker used in list
compact Says list should render smaller than usual
ol reversed Descending list order
start Define start value of list
type Define marker to be used by list
table align Alignment of table (left,center,right)
bgcolor Background color of table
width Width of table
form name Name of form
target Where to display response (e.g. _blank = new page)
accept File type that it accepts
button disabled Disable button (cannot click)
name Button name
value Set initial button value
