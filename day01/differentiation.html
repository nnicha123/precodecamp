tags to be compared   similarities   differences
hr&br Both creates break   hr -> break+line (change of content), br -> only break
ul&ol Both creates list   ul -> bullets, ol -> numbered
select&checkbox&radiobutton Both needs choice to be made   select -> dropdown list, checkbox -> square box that can be ticked, radio button -> circular can be highlighted
div&span Both used as content wrappers   div -> block-line (line break before & after), span -> in-line
script&style Both used as to style page   style -> calls css, script -> calls Javascript in page
