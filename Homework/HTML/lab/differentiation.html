Tags to be compared   Similarities   Differences
hr & br Both creates break (to the next line)   hr -> Has break & line (use when there is a change of content)
  br -> Only break, no line used only to go to next line
ul & ol Both creates list   ul -> Bullets (or other style) but not numbered/ordered
  ol -> Numbered (ordered list) used when list needs some kind of order
select & checkbox & radiobutton Both needs choice to be made   select -> Dropdown list to select from
  checkbox -> Square box that can be ticked to select
  radio button -> circular can be highlighted
div & span Both used as content wrappers   div -> Block-line (line break before & after)
  span -> Text in the same line (in-line text)
script & style Both used as to style page   style -> Calls css for styling e.g. border, color,size..
  script -> Calls Javascript in page, for functionality e.g. "On-click" of button
