input type   explanation
text-box   Blank box that accepts texts
check-box   Small-box that can be ticked to select
radio   Small-circle that can be highlighted to select
datetime-local   Can select & date and time, will come as a calendar choice (drop-down)
password   Accepts password in form of dots (hidden as default)
button   Format button that is clickable, can be linked to JS for functionality "on-click"
color   Can select color (from range)
email   Format of email accepted, drop-down of valid email available when click box
file   Format of a button use to browse files from server & select
image   Format of a button use to browse image from server & select
